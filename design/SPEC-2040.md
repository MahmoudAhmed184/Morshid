## 0. Concept

Keep the manuscript's soul — paper, ink, rubric red, ultramarine, Fraunces, footnotes. Re-materialize it as a **2040 spatial interface**: content lives on floating sheets above the paper, chrome is frosted glass, rounding is generous and continuous, depth is soft layered shadow (never glow), motion is fluid and physical. "A rare-books reading room, digitized by someone with taste."

### Kill list (unchanged unless noted)
Still banned: gradient text, multi-hue/saturated gradients, purple/indigo washes, star ornaments, fake logos/stats/avatars, colored glow shadows, emoji/sparkles iconography.
**Newly ALLOWED (only as specified):** monochrome paper-tone atmosphere (`.atmosphere`), frosted glass chrome (`.glass-paper`), hover float-lifts, large radii, layered soft shadows, once-only scroll reveals.

## 1. Token deltas (client/src/styles.css)

Palette (light + dark): UNCHANGED from v2 (Reading Room / Lamplight, rubric + ink-blue + semantic set). Only these change:

**Radius — continuous, generous:**
```
--radius: 1rem;
--radius-sm: 0.625rem;  --radius-md: 1rem;  --radius-lg: 1.25rem;
--radius-xl: 1.5rem;    --radius-2xl: 2rem; --radius-3xl: 2.75rem;
```
Usage law: buttons & badges = pill (`rounded-full`); inputs/controls `rounded-xl`; cards/dialogs/figures `rounded-2xl`; hero panels, auth ink panel, colophon band, sheets `rounded-3xl`; never below `rounded-lg` except tiny chips (`rounded-lg`).

**Shadows — warm ink, layered, real (ink = oklch(0.24 0.02 60)):**
```
--shadow-xs: 0 1px 2px ink/0.05;
--shadow-sm: 0 2px 8px ink/0.06, 0 1px 2px ink/0.04;      /* resting cards */
--shadow-md: 0 8px 24px ink/0.08, 0 2px 6px ink/0.05;     /* floating chrome, lifted cards */
--shadow-lg: 0 16px 48px ink/0.12, 0 4px 12px ink/0.06;   /* overlays, transcript sheet */
--shadow-xl: 0 24px 64px ink/0.16, 0 6px 16px ink/0.08;   /* marquee panels */
```
Dark theme: same geometry, black-based, roughly double alpha.

**New/changed utilities:**
- `.glass-paper` — `background: color-mix(in oklab, var(--background) 72%, transparent); backdrop-filter: blur(20px) saturate(1.4); border: 1px solid color-mix(in oklab, var(--foreground) 10%, transparent);`
- `.atmosphere` — monochrome radial wash for section backdrops: `background: radial-gradient(80% 60% at 50% 0%, var(--secondary), transparent 70%);` (paper-tone only; NEVER hued).
- `.sheet-stack` — floating-pages motif. Positions two paper sheets behind the element via ::before/::after: both `position:absolute; inset:0; z-index:-1; background:var(--card); border:1px solid var(--border); border-radius:inherit;` ::before `transform: rotate(-1.4deg) translate(-10px, 12px)`, ::after `transform: rotate(0.9deg) translate(12px, 18px); opacity:.7`. Parent needs `relative z-0`. Hidden below `md:`.
- `.hover-float` — `transition: transform .25s var(--ease-out-quart), box-shadow .25s var(--ease-out-quart);` `@media(hover) & motion-safe`: hover `transform: translateY(-3px)` + shadow-md.
- KEEP: `.rule`, `.rule-strong`, `.smallcaps-label`, `.footnote`, `.rubric-square`, `.link-editorial`, `.scrollbar-themed`, `.shimmer`, fade keyframes. Add `--ease-out-quart: cubic-bezier(0.25,1,0.5,1)` and `--ease-spring: cubic-bezier(0.34,1.4,0.64,1)` if absent.
- **Reveal component**: `client/src/components/reveal.tsx` — IntersectionObserver, plays `motion-safe:animate-fade-up` ONCE at 15% visibility, SSR-safe, renders children plainly under reduced motion / no IO. Used by landing sections only.

## 2. Primitives (client/src/components/ui/) — APIs stay backward compatible

- **Button**: pill `rounded-full`. default: `bg-foreground text-background hover:bg-primary hover:text-primary-foreground shadow-xs hover:shadow-sm motion-safe:hover:-translate-y-px active:translate-y-0`; sizes: sm h-8 px-4, default h-10 px-5, lg h-12 px-7 text-base. outline/secondary/ghost/destructive/link mirror this geometry.
- **Input/Textarea/Select trigger**: `rounded-xl bg-secondary/60 border border-transparent h-10 px-4; focus: bg-background border-input ring-2 ring-ring/35 (no offset)`. Error: `border-rubric ring-rubric/25`.
- **Card**: `rounded-2xl border-foreground/8 bg-card shadow-sm`. Interactive cards add `.hover-float`.
- **Badge**: pill, mono uppercase, soft washes (unchanged colors). Tiny chips `rounded-lg`.
- **Dialog/AlertDialog**: `rounded-3xl shadow-lg p-8`; overlay `bg-foreground/25 backdrop-blur-sm`.
- **Popover/Dropdown/Command/Tooltip**: `rounded-xl shadow-md bg-popover` (solid, not glass).
- **Sheet**: `rounded-l-3xl` (right side) / mobile menu `rounded-3xl m-3 glass-paper shadow-lg`.
- **Tabs**: segmented control — list `rounded-full bg-secondary/70 p-1`, triggers pill; active `bg-background shadow-xs`.
- **Skeleton**: `rounded-xl` (text lines `rounded-md`).
- **Sonner toasts**: `rounded-2xl glass-paper shadow-md`.
- Focus policy EVERYWHERE: `focus-visible:ring-2 ring-ring/60 ring-offset-2 ring-offset-background` (pill elements may drop offset). One convention, no ring-3.

## 7. Motion

Durations 200–350ms; `--ease-out-quart` default, `--ease-spring` for hover-float only. Once-only `<Reveal>` on landing sections. Staggered entrances (hero transcript, mobile menu) `motion-safe:`. Hover lifts only via `.hover-float` / Button spec. Still banned: loops, parallax, typing carets, scale > 1.02.

## 8. QA bars
